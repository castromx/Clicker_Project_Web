# Clicker Project

This is a clicker project similar to NotCoin.

## Technologies

### Backend:
- **Python**
- **FastAPI**
- **SQLAlchemy**
- **Redis**

### Frontend:
- **JavaScript**
- **React**
- **Axios**

## Latest Updates

In the last commit, I added a functioning navigation menu, which in the future will contain relevant information from databases.
